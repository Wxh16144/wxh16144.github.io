create.js使用分享

发表于 2019-04-20 更新于 2021-02-20

使用create.js包的时,如果你在浏览器直接使用报错那就对了

使用npm i create -D 安装的包,不经过打包无法使用,在浏览器是不能直接运行的;
如果你直接下载create.min.js 引入js的方式也报错,那就对了;
你需要使用 npm i createjs-npm -D;

好了说完了,你可以去调试了
