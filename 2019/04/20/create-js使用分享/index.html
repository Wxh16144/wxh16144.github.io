使用 create.js 包的时, 在浏览器直接使用会报错

使用 npm i create -D 安装的包, 不经过打包无法使用, 在浏览器是不能直接运行的;
如果你直接下载 create.min.js 引入 js 的方式也报错, 那就对了;
你需要使用 npm i createjs-npm -D;
