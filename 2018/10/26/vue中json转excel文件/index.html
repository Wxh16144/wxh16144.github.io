选项

  • 数据数据

    • fileName 下载文件名（默认：下载）

  • 表格选项

  1. sheetName sheet名字（可有可无）（默认sheet1）
  2. **sheetHeader **标题（excel第一行数据）
  3. columnWidths列宽非必须
  4. sheetData数据源（必须）
  5. sheetFilter 列过滤（只有在数据为对象下起作用）（可有可无）
